# Estate Square

This project covers various aspects of web development, including advanced authentication, real-world CRUD operations, user-friendly features, advanced search functionality, and deployment using the 'render' platform.

## Technologies Used:

🔑 **Advanced Authentication:** Implement JWT, Firebase, and Google OAuth for secure and seamless user access.

🏡 **Real-world CRUD Operations:** Create, read, update, and delete property listings using MongoDB.

💡 **User-friendly Features:** Enhance the user experience with image uploads, property listing management, and more.

🚀 **Advanced Search Functionality:** Implement cutting-edge search features to help users find what they're looking for.

🌐 **Deployment:** Explore the live website here! https://mern-estatesquare.onrender.com
